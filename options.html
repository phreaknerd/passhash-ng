Adjust the appearance and functionality of the popup.

The master-key-hint shows you a two character hash of your master-key in the input field. With this you can see if you typed your key correctly without having to unmasking it.

Storage

This extension is using a synced storage. In combination with a Firefox-Sync-account you can synchronize all settings (including the site-options) to other devices.

Learn more about Firefox Sync.

Hashing

These settings define how your passwords are hashed on a new site by default. This will only affect new sites on the first login.
Some sites are quite restrictive about which passwords are valid, so you can always adjust those settings per site in the hasher dialog. The changed values will be reloaded on your next visit.
